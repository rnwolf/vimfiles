# vimfiles

Portable vim configuration

## Install on windows

open Powershell prompt
```
cd ~
git clone git://github.com/rnwolf/vimfiles ~/vimfiles
mklink _vimrc vimfiles\vimrc
```
## Install on linux

```
cd ~
git clone git://github.com/rnwolf/vimfiles ~/.vim
ln -s ~/.vim/vimrc ~/.vimrc
```
